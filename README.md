# Ask-Me-Anything

Android Chat App to chat with Chat GPT bots of various kinds made by Jetpack Compose!


## Features

- Speak with 30+ bots or create your own
- Take instant answers
- Voice to Text
- Customizable chat design


## Technologies

- Jetpack Compose
- Retrofit
- Work Manager
- DataStore
- Room
- Clean Architecture
- Hilt/Dagger2
- Multi-Modules
- Accompanist
